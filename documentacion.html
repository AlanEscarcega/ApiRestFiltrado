API de Perros

Una API REST para gestión de perros, búsquedas, filtros y ordenamiento.

GET /api/perros/

Obtiene todos los perros registrados.

Request
GET http://127.0.0.1:8000/api/perros/
Response
{
    "message": "Perros obtenidos correctamente",
    "total": 5,
    "data": []
}
Status Codes
  • 200 OK
  • 400 Bad Request

GET /api/perros/1/

Obtiene un perro específico por ID.

Request
GET http://127.0.0.1:8000/api/perros/1/
Response
{
    "message": "Perro encontrado",
    "data": {}
}
Status Codes
  • 200 OK
  • 404 Not Found
