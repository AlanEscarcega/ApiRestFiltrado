Proyecto desarrollado con Django REST Framework para gestionar información de perros en un sistema de adopción.

## Funcionalidades
- Listar perros
- Buscar por ID
- Filtrar por raza
- Búsqueda por nombre
- Ordenamiento por edad y peso
- Filtrado de vacunados
- Endpoint personalizado (alta energía)

## Endpoints principales
- GET /api/perros/
- GET /api/perros/{id}/
- GET /api/perros/raza/{raza}/
- GET /api/perros/buscar/{nombre}/
- GET /api/perros/?ordering=edad
- GET /api/perros/vacunados/
- GET /api/perros/alta-energia/
